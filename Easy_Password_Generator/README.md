# Генератор паролей (Python)

Надёжный CLI-генератор случайных паролей на Python с использованием криптографически стойкого модуля secrets.  
Поддерживает настройку длины, состава символов, генерацию нескольких паролей, оценку энтропии и копирование в буфер обмена.

## Возможности
- Криптографически стойкая генерация (secrets)
- Настраиваемые классы символов: строчные, заглавные, цифры, символы
- Количество паролей за один запуск (-n/--count)
- Оценка энтропии в битах
- Копирование первого пароля в буфер обмена (--copy, требует pyperclip)
- Удобные флаги командной строки

## Установка

> Требуется Python 3.8+

1) (Рекомендуется) создать виртуальное окружение
```bash
python -m venv .venv
# Linux/Mac:
source .venv/bin/activate
# Windows:
.venv\Scripts\activate
```
2) Установить зависимости
```bash
pip install -r requirements.txt
```
3) Запуск
```bash
python password_generator.py [options]
```
Примеры:
```bash
# 1 пароль длиной 16 (по умолчанию), со всем набором символов
python password_generator.py

# 3 пароля длиной 24
python password_generator.py -l 24 -n 3

# Без символов (только буквы и цифры)
python password_generator.py --no-symbols

# Только строчные + цифры, длиной 20
python password_generator.py --no-upper --no-symbols -l 20

# Скопировать первый пароль в буфер обмена (если установлен pyperclip)
python password_generator.py --copy
```
Опции CLI:
```bash
-l, --length INT       Длина пароля (по умолчанию 16)
--no-lower             Исключить строчные буквы
--no-upper             Исключить заглавные буквы
--no-digits            Исключить цифры
--no-symbols           Исключить символы
-n, --count INT        Сгенерировать N паролей (по умолчанию 1)
--copy                 Скопировать первый пароль в буфер обмена (требуется pyperclip)
-h, --help             Показать справку
```
